# graph_generator.py

A simple python tool for linux to generate graphs using the Matplotlib. 
Purpose was to create a simple way of generating graphs using one liners in the terminal

Examples:

 # Data for the y-axis
 DATA="10,8,15,17,1"
 python graph_generator.py --data-set "$DATA"
